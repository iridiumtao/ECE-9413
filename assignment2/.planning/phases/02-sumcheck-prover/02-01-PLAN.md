## Algorithm

```python
def sumcheck_32(eval_tables, *, q, expression, challenges, num_rounds):
    """Compulsory 32-bit sumcheck path."""
    # Degree of the round polynomial = length of the longest multiplicative term.
    # Examples: [["a"]] -> 1, [["a","b"]] -> 2, [["a","b"],["c"]] -> 2, [["a","b","c"]] -> 3
    degree = max(len(term) for term in expression)

    # Work on mutable copies of the tables so folds do not mutate the input dict.
    # eval_tables values are jnp.uint32 arrays of length 2^num_vars.
    tables = {var: jnp.asarray(table, dtype=jnp.uint32)
              for var, table in eval_tables.items()}

    round_evals_list = []

    for i in range(num_rounds):
        # ---------------------------------------------------------------
        # Step 1: Evaluate round polynomial g_i at each point v in {0, ..., degree}.
        # g_i(v) = sum over all remaining Boolean entries of the expression,
        #          where the current variable is fixed to v via MLE extension.
        #
        # Indexing convention (D-02 — LOCKED):
        #   even indices (0, 2, 4, ...) -> current variable = 0
        #   odd indices  (1, 3, 5, ...) -> current variable = 1
        #   table_at_v = mle_update_32(table[::2], table[1::2], v, q=q)
        #
        # For each additive term in expression (each inner list of variable names):
        #   1. For each variable name in the term: compute table_at_v using its table.
        #   2. Multiply these arrays elementwise using mod_mul_32.
        #   3. Sum the result across all table entries using uint64 reduction.
        # Then sum across all additive terms mod q.
        # ---------------------------------------------------------------
        g_i_evals = []

        for v in range(degree + 1):
            # v is a Python integer (0, 1, ..., degree).
            # Convert to uint32 scalar for mle_update_32.
            v_scalar = jnp.asarray(v, dtype=jnp.uint32)

            g_i_at_v = jnp.asarray(0, dtype=jnp.uint64)

            for term in expression:
                # Each term is a list of variable names, e.g. ["a", "b"].
                # Start with the first variable's MLE at v, then multiply in the rest.
                first_var = term[0]
                term_product = mle_update_32(
                    tables[first_var][::2],
                    tables[first_var][1::2],
                    v_scalar,
                    q=q,
                )  # shape: (current_table_size // 2,), dtype uint32

                for var in term[1:]:
                    factor = mle_update_32(
                        tables[var][::2],
                        tables[var][1::2],
                        v_scalar,
                        q=q,
                    )
                    term_product = mod_mul_32(term_product, factor, q)

                # Safe uint64 sum: table has at most 2^20 entries, each < q < 2^32,
                # so sum < 2^52 < 2^64 — no overflow.
                term_sum = jnp.sum(term_product.astype(jnp.uint64)) % jnp.asarray(q, dtype=jnp.uint64)
                g_i_at_v = (g_i_at_v + term_sum) % jnp.asarray(q, dtype=jnp.uint64)

            g_i_evals.append(jnp.asarray(g_i_at_v, dtype=jnp.uint32))

        # Stack the degree+1 scalar evaluations into a 1D array for this round.
        round_evals_list.append(jnp.stack(g_i_evals))

        # ---------------------------------------------------------------
        # Step 2: Fold all tables using challenges[i] (D-07 — LOCKED).
        # Applied only for the first num_rounds - 1 rounds.
        # ALL tables in eval_tables are folded, even if not in expression.
        # ---------------------------------------------------------------
        if i < len(challenges):
            r = challenges[i]
            tables = {
                var: mle_update_32(tbl[::2], tbl[1::2], r, q=q)
                for var, tbl in tables.items()
            }

    # Stack per-round arrays into shape (num_rounds, degree+1), dtype uint32.
    round_evals = jnp.stack(round_evals_list)  # shape: (num_rounds, degree+1)

    # claim0 = g_0(0) + g_0(1) mod q (D-04 — LOCKED).
    # Mathematically equal to the sum over all 2^n Boolean inputs.
    claim0 = mod_add_32(round_evals_list[0][0], round_evals_list[0][1], q)

    return claim0, round_evals
```

Key points:
- `degree = max(len(term) for term in expression)` — handles all base expressions.
- Table slicing `table[::2]` / `table[1::2]` follows LSB-first interleaved convention (D-02).
- `mle_update_32` is used for all `v`, including v=0, v=1, and v=2,3,... (D-03/D-06).
- Tables dict is rebuilt each fold round, not mutated in-place.
- `challenges` has length `num_rounds - 1`; the `if i < len(challenges)` guard ensures
  the last round skips folding (D-07).
- `claim0` is a scalar `jnp.uint32` derived from round 0 evals (D-04).
- `round_evals` shape is `(num_rounds, degree+1)` (D-08).

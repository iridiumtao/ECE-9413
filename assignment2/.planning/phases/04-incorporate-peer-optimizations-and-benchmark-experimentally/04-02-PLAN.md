<objective>
Extend the t=0/t=1 shortcut from Plan 01 by pre-computing `evens` and `odds` dicts once per round. Currently, each reference to `tables[var][::2]` or `tables[var][1::2]` re-slices the array. After this change, both the shortcut branches and the `mle_update_32` call for v >= 2 read from pre-computed dicts. The fold step is also updated to use `evens[var]`/`odds[var]` instead of re-slicing inline.

Purpose: Remove redundant array slicing operations. For an expression with N terms and a degree-d polynomial, the current code slices each table `(d+1) * len(term)` times per round plus once more for the fold. After this change: always 1 slice per variable per round.

Output: Modified `student.py`, passing tests, Experiment 02 section added and filled in `experiment.md`.

    for i in range(num_rounds):
        evens = {var: tbl[::2]  for var, tbl in tables.items()}
        odds  = {var: tbl[1::2] for var, tbl in tables.items()}
        g_i_evals = []

        for v in range(degree + 1):
            g_i_at_v = jnp.asarray(0, dtype=jnp.uint64)

            for term in expression:
                if v == 0:
                    term_product = evens[term[0]]
                    for var in term[1:]:
                        term_product = mod_mul_32(term_product, evens[var], q)
                elif v == 1:
                    term_product = odds[term[0]]
                    for var in term[1:]:
                        term_product = mod_mul_32(term_product, odds[var], q)
                else:
                    v_scalar = jnp.asarray(v, dtype=jnp.uint32)
                    term_product = mle_update_32(
                        evens[term[0]], odds[term[0]], v_scalar, q=q,
                    )
                    for var in term[1:]:
                        factor = mle_update_32(evens[var], odds[var], v_scalar, q=q)
                        term_product = mod_mul_32(term_product, factor, q)

                # Safe uint64 sum: up to 2^20 entries, each < q < 2^32, sum < 2^52 < 2^64.
                term_sum = jnp.sum(term_product.astype(jnp.uint64)) % jnp.asarray(q, dtype=jnp.uint64)

        round_evals_list.append(jnp.stack(g_i_evals))

        # Fold all tables using challenges[i]; reuses pre-computed evens/odds.
